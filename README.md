## üîç What is ARBE Œª*?

**ARBE Œª*** (Absorption‚ÄìReflection Balance Edge) is a physically grounded scalar that describes the balance point between absorption and reflection of a colour sample in the visible spectrum.

- Computed from spectral reflectance data `R(Œª)` (typically from 380 to 730‚ÄØnm)
- Expressed in **nanometres (nm)** as a single index: **Œª\***
- Integrates with **CIELAB** and **HLC** colour spaces
- Designed for **quality control**, **design**, and **pigment development**

> Œª\* is defined as the **first rising edge crossing** at  
> `R(Œª) = ‚àö2 ‚àí 1 ‚âà 0.4142`  
> using **linear interpolation**.
